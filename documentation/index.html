Getting Started

To start, you need to recall the compressed script of Duedo on the page. Next you will need to create a canvas element and use it to create the application instance.

Set the html and initialize

Create a canvas and specify the size that the canvas will have, and therefore the viewport of your application

Our you can use the grid system with 2 columns like this;

                                    /*
                                     * Create a new instance, specifying the canvas to use, 
                                     * the size in length and size in height of the virtual world.
                                    */
                                    app = new Duedo.GameContext(document.getElementById("screen"), 5500, 1000);

If everything was fine, you should see the splash screen.

From now on we will refer to the application instance as "app".

Load the resources and use them

To load and use resources you need an active server. You can use MAMP or Node or use a remote server via FTP.

The Loader object

app.Loader

There is no application without some kind of resource, images or audio. To add resources to the application you must first load them and make them available to the application that will make immediate use of them. For this reason, the app object provides the Loader that will load and provide the resource.
To access the Loader just call it up 

                            const loader = app.Loader;

Then you can immediately use it to load resources that are present at the specified address. You can upload images, audio, json, spritesheet.

                            //Preload a jpg
                            loader.AddResource("media/ball.jpg");

                            // If you want to specify another name for the resource, pass it as second parameter
                            loader.AddResource("media/music.wav", "soundtrack");

                            // In this case I am specifying two variant formats of the same audio to meet the possible 
                            // incompatibilities given by the various browsers.
                            loader.AddResource("media/click.wav | media/click.mp3");

                            // Load a json
                            Game.Loader.AddResource("src/anims/frames.json", 'frames');

                            // Bind the "complete" callback function
                            loader.Bind("complete", completeCallback);

                            // Start loading process
                            loader.StartLoading();

When the loading has been completed, the specified callback will be called, within which it is possible to start all the processes that will exploit those certain resources. So, in this case, the application we are developing.

Using the resources

Image

The "image" entity represents the most classic object of an interactive application. It allows you to create an image object starting from a resource that has been preloaded through the Loader. It belongs to the category of Graphic Objects, and therefore has a series of properties, first of all the position in space, represented by a 2d vector Duedo.Vector2.
Lets create it. 

                            // Create a "ball" image
                            const ball = new Duedo.Image(game, game.Cache.GetImage("ball"));
                            ball.Width = 30;
                            ball.Height = 30;
                            // Location is based on the center of the object, unless you set CenterRelative = false
                            ball.Location.X = 15;
                            ball.Location.Y = 15;

Once you created the image object, put it in the application's world with:

                            app.Add(ball);

From now on you will see your ball appear on the canvas at the position set by the Location vector.

Sound

Duedo offers the Sound and SoundManager objects. Through the Sound object it is possible to play an audio document, stop it, assign a location in the Cartesian space (and its volume will vary according to the distance from the viewport), repeat it a few times or infinitely, modify its volume.

Be careful, browsers accept different audio formats depending on the manufacturer. Please inquire before attempting to run the file. See compatibilities.

To run a file that has been loaded, you can do it with:

                            // Return a Sound object
                            const mySound = app.SoundManager.Get("soundtrack");
                            mySound.Play();

Other methods are

Method
Description
Pause()
Pause a playing audio
Stop()
Stop a playing audio
Resume
Resume a paused audio
SetLocation(x, y)
Position the sound in the Cartesian plane. Its volume varies according to the distance from the Viewport, making it more audible or less audible. For instance, the sound of an object in the world
SetMaxDistance(n: Number)
If the sound has a location, you can assign a maximum distance limit, so that the sound volume is never 0
SetVolume(n: Number)
Set the volume of a static sound (for instance, a soundtrack)
Loop(value: Boolean)
Decide if the sound will repeat indefinitely

To attach callbacks you can use Bind

Spritesheet

Spritesheets are a classic of 2d animation. To be able to use them there is a dedicated object called Spritesheet.

                            // Create a spritesheet from a preloaded image
                            const runner = new Duedo.SpriteSheet(app, app.Cache.GetImage("runningGrant"));
                            // Set location
                            runner.Location.X = 350;
                            runner.Location.Y = 20;
                            // Create the frame sequence
                            runner.AddSequence("runright", [
                                [1,70, 158,213], // Each line correspond to one frame
                                [344,70,147,217],
                                [685,73,116,217],
                                [1025,72,106,218],
                                [1363,67,98,224],
                                [1666,67,143,224],
                                [11,359,152,225],
                                [508,362,148,209],
                                [845,363,140,209],
                                [1192,364,117,219],
                                [1532,363,96,221],
                                [1848,359,108,224],
                                [18,906,138,218]
                            ]);
                            // Play the sequence
                            const sequence = runner.PlaySequence("runright");
                            app.Add(runner);
